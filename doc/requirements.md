**项目名称**: AI Voice Draw  
**版本**: v1.0  
**日期**: 2026-06-13  
**状态**: 需求规划阶段

---

## 1. 项目概述

### 1.1 项目目标
开发一个纯语音交互的AI绘图工具，用户无需使用鼠标或键盘，仅通过语音指令完成绘图创作。

### 1.2 核心价值
- **无障碍创作**: 为视障人群、肢体不便用户提供绘图能力
- **自然交互**: 通过自然语言描述实现创意表达
- **快速原型**: 通过语音快速生成视觉原型

### 1.3 技术架构
- **前端**: Vue 3 + Web Speech API (语音识别与合成)
- **后端**: FastAPI (Python)
- **AI服务**: 阿里云百炼平台
  - 推理模型: qwen-3.7plus (指令理解与画面状态管理)
  - 生图模型: qwen-image (图像生成)
- **部署**: 无需登录、无数据库、无session管理

## 2. 功能需求

### 2.1 核心绘图模式

#### 2.1.1 智能生成模式 (推荐实现)
**描述**: 用户描述想要的内容，AI生成图像并智能放置在画布上

**语音指令示例**:
- "生成一个红色的太阳"
- "在左上角添加一朵云"
- "画面中央放一棵树"

**技术实现**:
```
用户语音 → Web Speech API识别 → qwen-3.7plus理解意图+画布状态
         → 生成prompt → qwen-image生图 → 前端Canvas合成
```

**状态管理**:
- 画布对象列表: `{id, name, description, position, size, zIndex, imageUrl}`
- 每次新增对象，更新上下文prompt，包含现有所有对象信息
- qwen-3.7plus基于当前画布状态，决定新对象的合理位置

**可行性**: ✅ 高
**实现优先级**: P0 (核心功能)

---

#### 2.1.2 快速调整模式 (推荐实现)
**描述**: 生成图像后，通过语音调整其属性

**语音指令示例**:
- "把太阳放大一点"
- "把云朵向右移动"
- "旋转这棵树30度"
- "让花朵半透明"
- "确认" / "取消"

**可调整属性** (前端实现):
- ✅ 位置 (x, y坐标或相对方位)
- ✅ 大小 (缩放比例)
- ✅ 旋转 (角度)
- ✅ 透明度
- ✅ 层级 (置于顶层/底层)

**不支持属性** (需重新生图，暂不实现):
- ❌ 样式变化 ("把太阳变成卡通风格")
- ❌ 形状变化 ("把圆形太阳变成方形")
- ❌ 颜色变化 ("把红色太阳变成黄色") - 可在v2.0考虑重新生图实现

**交互流程**:
1. 用户语音调整指令
2. 前端实时预览调整效果 (虚线框显示)
3. 用户语音"确认"应用，或"取消"还原

**可行性**: ✅ 高 (仅限前端可控属性)
**实现优先级**: P0 (核心功能)

---

#### 2.1.3 几何图形模式 (可选实现)
**描述**: 替代原"自由绘制模式"，语音绘制预定义几何图形

**原因**: 语音控制任意曲线轨迹不可行，改为绘制标准图形

**语音指令示例**:
- "用红色画笔画一条直线，从左上到右下"
- "画一个蓝色圆形，在画面中央，半径50像素"
- "画一个绿色矩形，左上角坐标(100, 100)，宽200高150"

**支持图形**:
- 直线 (起点、终点、颜色、粗细)
- 圆形 (圆心、半径、填充色、边框色、边框粗细)
- 矩形 (位置、宽高、填充色、边框色、边框粗细)
- 椭圆 (中心、长轴、短轴、填充色、边框色)
- 文字 (内容、位置、字体、大小、颜色)

**技术实现**: Canvas 2D API直接绘制，无需AI生图

**可行性**: ✅ 高
**实现优先级**: P1 (增强功能)

### 2.3 语音交互

#### 2.3.1 语音输入
**技术**: Web Speech API (SpeechRecognition)

**功能**:
- ✅ 持续监听模式 (可随时说话)
- ✅ 按键激活模式 (按住空格键说话，松开识别)

**容错机制**:
- 识别结果置信度低时，语音询问确认

**实现优先级**: P0

**实现优先级**: P0

## 3. 指令系统设计

### 3.1 指令分类

| 类别 | 指令示例 | 处理方式 | 优先级 |
|------|----------|----------|--------|
| **创建对象** | "画一个红色的太阳" | qwen-3.7plus解析 → qwen-image生图 | P0 |
| **调整对象** | "把太阳放大两倍" | qwen-3.7plus解析 → 前端CSS transform | P0 |
| **删除对象** | "删除太阳" | qwen-3.7plus识别目标 → 前端移除 | P0 |
| **画布操作** | "清空画布" | 关键词匹配 → 前端执行 | P0 |
| **历史操作** | "撤销" | 关键词匹配 → 历史栈 | P0 |
| **查询状态** | "画布上有什么" | qwen-3.7plus生成描述 | P1 |
| **保存导出** | "保存作品" | 关键词匹配 → 前端导出 | P0 |
| **系统控制** | "关闭语音播报" | 关键词匹配 → 配置更新 | P1 |

### 3.2 指令解析流程

```
用户语音
  ↓
Web Speech API识别
  ↓
识别文本
  ↓
前端预处理 (关键词快速匹配: "撤销"、"保存"等)
  ↓ (未匹配到快速指令)
发送到后端 FastAPI
  ↓
qwen-3.7plus理解意图
  - 输入: 识别文本 + 当前画布状态JSON
  - 输出: 结构化指令JSON
  ↓
后端执行指令
  - 生图类: 调用qwen-image → 返回图片URL
  - 调整类: 返回调整参数
  - 查询类: 返回查询结果
  ↓
前端应用结果
```

### 3.3 指令格式规范 (qwen-3.7plus输出)

```json
{
  "intent": "create_object | adjust_object | delete_object | query | canvas_op",
  "target": "object_id or object_name",
  "action": "具体操作",
  "parameters": {
    "position": {"x": 100, "y": 200},
    "size": {"width": 150, "height": 150},
    "rotation": 30,
    "opacity": 0.8,
    "zIndex": 5,
    "prompt": "a red sun in cartoon style"  // 用于生图
  },
  "confirmation_needed": false
}
```

## 4. 技术约束与限制

### 4.1 已知限制

| 限制项 | 说明 | 影响 |
|--------|------|------|
| **语音延迟** | Web Speech API识别 + qwen推理 + qwen-image生图 ≈ 3-8秒 | 无法实现实时绘制 |
| **图像编辑能力** | qwen-image仅支持文生图，不支持图生图编辑 | 无法实时调整图像样式/颜色 |
| **精确坐标控制** | 语音难以表达精确像素坐标 | 采用相对位置描述 ("左上角"、"中央") |
| **复杂曲线** | 语音无法描述任意贝塞尔曲线 | 不支持自由曲线，仅支持几何图形 |
| **无状态后端** | 不使用数据库/session | 画布状态由前端管理，每次请求携带完整状态 |

### 4.2 浏览器兼容性

- **Web Speech API**: Chrome/Edge 支持良好，Firefox/Safari 部分支持
- **推荐浏览器**: Chrome 90+ / Edge 90+
- **不支持**: IE 11及以下

## 5. 非功能需求

### 5.1 性能要求
- 语音识别响应 < 1秒
- 指令解析响应 < 2秒
- 图像生成响应 < 5秒 (qwen-image限制)
- 前端调整操作 < 100ms

### 5.2 可用性要求
- 语音识别准确率 > 85% (中文)
- 指令理解准确率 > 90%
- 操作成功率 > 95%

### 5.3 可访问性
- 完全支持纯语音操作
- 提供视觉辅助反馈 (文字显示识别结果)
- 支持键盘快捷键作为备选 (撤销、保存等)
