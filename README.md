# AI Voice Draw - AI语音绘图工具

> 纯语音交互的AI绘图工具，无需鼠标和键盘，用自然语言描述即可完成绘图创作。

---

## 📖 项目简介

AI Voice Draw 是一个创新的语音绘图工具，用户通过语音指令即可完成绘图创作，无需使用鼠标或键盘。项目旨在为视障人群、肢体不便用户提供绘图能力，同时也为所有用户提供更自然的创意表达方式。

### 核心特性

- 🎤 **纯语音交互** - 完整的语音输入输出，支持中文识别和TTS播报
- 🎨 **AI智能生成** - 基于阿里云百炼平台的qwen-image模型自动生成图像
- 🔄 **实时调整** - 语音调整对象位置、大小、旋转、透明度
- ↩️ **撤销重做** - 完整的历史记录管理
- 💾 **快速导出** - 一句话保存作品为PNG/JPG格式
- 🚀 **即开即用** - 无需登录注册，打开网页即可使用

## 🛠️ 技术栈

### 前端
- **框架**: Vue 3 (Composition API)
- **状态管理**: Pinia
- **语音交互**: Web Speech API (SpeechRecognition + SpeechSynthesis)
- **画布渲染**: Canvas API
- **HTTP请求**: Axios

### 后端
- **框架**: FastAPI
- **数据验证**: Pydantic
- **异步HTTP**: httpx
- **AI服务**: 阿里云百炼平台
  - **推理模型**: qwen-3.7plus (指令理解)
  - **生图模型**: qwen-image (图像生成)

### 架构特点
- ✅ 前后端分离
- ✅ 无状态后端（无需数据库/session）
- ✅ 模块化设计（便于测试和维护）
- ✅ RESTful API

## 🚀 快速开始

### 前置要求
- Node.js 18+
- Python 3.11+
- 阿里云百炼平台API密钥
- Chrome/Edge浏览器（支持Web Speech API）

### 后端启动

```bash
cd backend

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入你的阿里云API密钥

# 启动服务
uvicorn app.main:app --reload --port 8000
```

后端服务运行在 `http://localhost:8000`

### 前端启动

```bash
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

前端服务运行在 `http://localhost:5173`

### 浏览器访问

打开 Chrome/Edge 浏览器，访问 `http://localhost:5173`，点击"开始监听"按钮，开始语音创作！
